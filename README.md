# Веб-приложение "Моменты жизни"

* Бэкенд: Django REST Framework, JWT
* Фронтенд: React, Redux Toolkit, Redux Saga, Sass
* База данных: PostgreSQL
* S3: Minio
* Другие используемые технологии: Docker, nginx
